   - Privacy: approximate area

2. Image context:
   - Output: bite-like mark
   - Confidence: medium
   - Disclaimer: this is not a diagnosis

3. Risk context:

   - Weather conditions may support mosquito activity
   - Similar reports rising nearby
   - Several bite-like image categories in the same time window

4. Personal result:
   - Signal: Moderate
   - Main contributors: symptom similarity, bite-like image category, outdoor exposure, nearby community trend

5. GeoSignal Map:
   - Sabino Canyon / Catalina Foothills zone moves to cluster watch

6. Public health dashboard:
   - Northeast Tucson outdoor/vector-like reports increasing

7. Human review queue:
